# Odd-Even Calculator

An interactive web-based calculator built with **HTML, CSS, and JavaScript** that checks whether a given number is **even or odd** using an `if...else` statement.

---

## 📌 Features
- Simple and user-friendly interface
- Input field for entering numbers
- Button to check whether the number is even or odd
- Dynamic result display with styled feedback


## 🛠️ Technologies Used
- **HTML** → Structure of the webpage
- **CSS** → Styling and layout
- **JavaScript** → Logic for odd/even calculation

---

## 🚀 How to Run
1. Clone or download this repository.
2. Make sure the following files are present:
   - `index.html`
   - `style.css`
   - `script.js`
3. Open `index.html` in your browser.
4. Enter a number and click **Check**.
5. The result will show whether the number is **Even** or **Odd**.
